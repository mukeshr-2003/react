# Express-Pug Employee Management

A simple Employee Management web application built with **Express.js**, **Pug**, and **MongoDB**.

---

## Features

- List all employees
- Add new employees
- Edit existing employees
- Delete employees

---

## Technologies Used

- **Node.js**  
- **Express.js**  
- **Pug** (Template engine)  
- **MongoDB** (Database)  
- **Mongoose** (MongoDB ODM)
